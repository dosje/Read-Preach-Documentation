# Installation

## Voraussetzungen

- **iPhone oder iPad**
- **iOS 16.0 oder neuer**
- Nach der Installation ist keine Internetverbindung erforderlich

## Installation aus dem App Store

1. Öffne den **App Store** auf deinem iPhone oder iPad
2. Suche nach **"Read & Preach"**
3. Tippe auf **Laden**, dann authentifiziere mit Face ID, Touch ID oder deinem Apple-ID-Passwort
4. Die App wird heruntergeladen und erscheint auf deinem Startbildschirm

## App aktualisieren

1. Öffne den **App Store**
2. Tippe auf dein Profilbild oben rechts
3. Scrolle nach unten, um verfügbare Updates zu sehen
4. Tippe auf **Aktualisieren** neben Read & Preach

Alternativ aktiviere automatische Updates unter **Einstellungen → App Store → App-Updates**.

## Daten auf ein neues Gerät übertragen

Read & Preach speichert alle Daten lokal. So überträgst du deine Daten:

1. Auf dem **alten Gerät**: Read & Preach öffnen → **Einstellungen → Export & Import**
2. Tippe auf **Export**, wähle was enthalten sein soll, und speichere die Datei (z. B. in iCloud Drive)
3. Auf dem **neuen Gerät**: Read & Preach aus dem App Store installieren
4. Öffne **Einstellungen → Export & Import → Import**
5. Wähle die exportierte Datei — alle Daten werden wiederhergestellt

## Deinstallation

Halte das Read & Preach-Symbol auf dem Startbildschirm gedrückt → **App entfernen** → **App löschen**.

> **Wichtig:** Durch das Deinstallieren werden alle Daten dauerhaft vom Gerät gelöscht. Exportiere deine Daten vorher, wenn du sie behalten möchtest.
