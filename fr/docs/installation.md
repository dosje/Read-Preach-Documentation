# Installation

## Prérequis

- **iPhone ou iPad**
- **iOS 16.0 ou version ultérieure**
- Aucune connexion Internet requise après l'installation

## Installation depuis l'App Store

1. Ouvrez l'**App Store** sur votre iPhone ou iPad
2. Recherchez **"Read & Preach"**
3. Appuyez sur **Obtenir**, puis authentifiez-vous avec Face ID, Touch ID ou votre mot de passe Apple ID
4. L'application se télécharge et apparaît sur votre écran d'accueil

## Mettre à jour l'application

1. Ouvrez l'**App Store**
2. Appuyez sur votre icône de profil en haut à droite
3. Faites défiler vers le bas pour voir les mises à jour disponibles
4. Appuyez sur **Mettre à jour** à côté de Read & Preach

Vous pouvez également activer les mises à jour automatiques dans **Réglages → App Store → Mises à jour des apps**.

## Transférer les données vers un nouvel appareil

Read & Preach stocke toutes les données localement. Pour transférer vos données :

1. Sur votre **ancien appareil** : ouvrez Read & Preach → **Paramètres → Export & Import**
2. Appuyez sur **Exporter**, choisissez ce qui doit être inclus, et enregistrez le fichier (par ex. dans iCloud Drive)
3. Sur votre **nouvel appareil** : installez Read & Preach depuis l'App Store
4. Ouvrez **Paramètres → Export & Import → Importer**
5. Sélectionnez votre fichier exporté — toutes les données seront restaurées

## Désinstallation

Appuyez longuement sur l'icône Read & Preach sur votre écran d'accueil → **Supprimer l'app** → **Supprimer l'app**.

> **Important :** La désinstallation supprime définitivement toutes les données de votre appareil. Exportez vos données au préalable si vous souhaitez les conserver.
