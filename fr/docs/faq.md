# Foire aux questions

## Général

**Qu'est-ce que Read & Preach ?**
Read & Preach est une application iOS native pour les frères qui combine le suivi des rapports de ministère et la progression de lecture biblique. Elle fonctionne entièrement hors ligne et ne nécessite aucun compte.

**L'application est-elle gratuite ?**
Oui. Read & Preach est entièrement gratuite — sans achats intégrés ni abonnement.

**Sur quels appareils l'application est-elle disponible ?**
iPhone et iPad uniquement (iOS 16.0 ou version ultérieure). L'application est optimisée pour les deux appareils avec une mise en page adaptative.

**Une connexion Internet est-elle nécessaire ?**
Non. L'application fonctionne entièrement hors ligne. Une connexion Internet n'est utilisée que si vous appuyez sur un lien pour ouvrir un passage sur JW.org ou dans JW Library.

**Quelles langues sont prises en charge ?**
Anglais, allemand et français. Changez la langue dans **Paramètres → Général → Langue**.

## Données & Confidentialité

**Où mes données sont-elles stockées ?**
Toutes les données sont stockées uniquement localement sur votre appareil. Rien n'est envoyé à des serveurs ou à des tiers.

**L'application collecte-t-elle des données personnelles ou des analyses ?**
Non. Read & Preach ne collecte, ne suit et ne partage aucune donnée.

**Comment sauvegarder mes données ?**
Utilisez la fonction d'export dans **Paramètres → Export & Import**. Vous pouvez exporter les rapports de ministère, la progression de lecture biblique (y compris la série et l'historique) et la lecture du Mémorial en fichier JSON.

**Comment restaurer mes données après une réinstallation ou sur un nouvel appareil ?**
Utilisez **Paramètres → Export & Import → Importer** et sélectionnez votre fichier JSON exporté précédemment.

**Que se passe-t-il si je supprime l'application ?**
Toutes les données sur votre appareil sont définitivement supprimées. Exportez vos données au préalable.

## Onglet Rapport

**Comment définir mon objectif mensuel d'heures ?**
Allez dans **Paramètres → Paramètres de rapport → Rôle récurrent**, sélectionnez votre rôle et définissez le mois de début.

**Puis-je saisir des données pour des jours passés ?**
Oui. Appuyez sur n'importe quel jour dans le calendrier pour basculer vers ce jour et saisir ou modifier des données.

**Qu'est-ce qu'une mission ponctuelle ?**
Une mission ponctuelle vous permet de définir un mois de pionnier auxiliaire temporaire (15 ou 30 heures) sans modifier votre rôle récurrent.

## Onglet Bible

**Comment configurer un plan de lecture ?**
Ouvrez **Paramètres → Paramètres de lecture biblique**, choisissez un mode de lecture et une durée, puis appuyez sur **Créer le plan**.

**Quels modes de lecture sont disponibles ?**
- **Séquentiel** — Genèse à Apocalypse
- **NT → AT** — Nouveau Testament d'abord, puis Ancien Testament
- **Personnalisé** — glissez les livres dans votre ordre préféré
- **Libre** — pas de planning, marquez les chapitres à votre rythme

**Que se passe-t-il si je manque un jour dans mon plan de lecture ?**
Les chapitres manqués sont automatiquement replanifiés sur les jours suivants tout en maintenant le rythme quotidien cible.

**Comment fonctionne la série ?**
La série compte les jours consécutifs pendant lesquels au moins un chapitre a été lu. Une **période de grâce de 2 jours** s'applique — si vous manquez un jour, la série ne se réinitialise pas immédiatement. Vous recevrez une notification de rappel le premier et le deuxième jour manqué. La série se réinitialise le troisième jour manqué.

**Comment voir quels chapitres j'ai déjà lus et quand ?**
Appuyez sur **Voir l'historique** dans la carte de progression. L'historique affiche un journal quotidien de tous les chapitres lus ainsi qu'un journal de chaque fois que vous avez terminé toute la Bible.

**Qu'est-ce que le Quiz de lecture ?**
Après avoir lu les chapitres du jour, un bouton Quiz apparaît dans la section Lecture du jour. Il présente des questions de compréhension basées sur les chapitres lus. Le quiz fonctionne dans tous les modes de lecture, y compris le mode libre.

**Puis-je modifier la difficulté du quiz ?**
Oui. Allez dans **Paramètres → Paramètres de lecture biblique → Difficulté du quiz** et choisissez Mixte (tous niveaux), Facile, Moyen ou Difficile. Le paramètre s'applique immédiatement.

**Combien de questions contient le quiz ?**
Vous pouvez définir le nombre maximum de questions par session dans **Paramètres → Paramètres de lecture biblique → Nombre max. de questions** (1 à 10).

**Qu'est-ce que le Suivi du Mémorial ?**
Le Suivi du Mémorial apparaît pendant la saison du Mémorial (environ 7 jours avant le 14 Nisan). Il affiche les passages bibliques assignés et les chapitres du livre *Jésus — le chemin, la vérité, la vie* pour chaque jour. Il se réinitialise automatiquement chaque année après la date du Mémorial.

## Dépannage

**L'application affiche des chapitres de Jérémie alors que je viens de commencer la Genèse.**
Cela peut se produire s'il existe un ancien plan de lecture. Allez dans **Paramètres → Paramètres de lecture biblique** et appuyez sur **Recalculer le plan**.

**Ma série s'est réinitialisée même si j'ai lu hier.**
La série permet jusqu'à 2 jours manqués avant de se réinitialiser. Si elle s'est réinitialisée, cela signifie que 3 jours ou plus consécutifs se sont écoulés sans qu'un chapitre soit marqué comme lu.

**La notification de rappel quotidien n'affiche pas les chapitres du jour.**
Assurez-vous d'avoir accordé l'autorisation de notifications à l'application dans **Réglages iPhone → Notifications → Read & Preach**. Vérifiez également que le rappel est activé dans **Paramètres → Paramètres de lecture biblique → Rappel quotidien**.
