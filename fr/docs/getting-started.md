# Premiers pas

Bienvenue sur Read & Preach ! Ce guide vous accompagne dans la configuration initiale.

## Prérequis

- **iPhone ou iPad** sous iOS 16.0 ou version ultérieure
- Aucun compte, aucune connexion Internet requise

## Étape 1 — Installer l'application

Téléchargez Read & Preach depuis l'App Store. Consultez le [guide d'installation](../installation/) pour plus de détails.

## Étape 2 — Configurer un plan de lecture (onglet Bible)

Lors de la première ouverture de l'onglet Bible, vous serez invité à configurer un plan de lecture :

1. Appuyez sur **Aller aux paramètres** (ou ouvrez directement l'onglet Paramètres)
2. Faites défiler jusqu'à **Paramètres de lecture biblique**
3. Choisissez un **mode de lecture** :
   - **Séquentiel** — Genèse à Apocalypse
   - **NT → AT** — Nouveau Testament d'abord, puis Ancien Testament
   - **Personnalisé** — organisez les livres dans votre propre ordre
   - **Libre** — marquez les chapitres lus sans planning quotidien
4. Choisissez une **durée** (1, 2 ou 3 ans)
5. Appuyez sur **Créer le plan**

## Étape 3 — Configurer votre rôle de ministère (onglet Rapport)

Pour que l'objectif mensuel d'heures s'affiche correctement :

1. Ouvrez l'**onglet Paramètres**
2. Faites défiler jusqu'à **Paramètres de rapport → Rôle récurrent**
3. Sélectionnez votre rôle : **Proclamateur**, **Pionnier auxiliaire** ou **Pionnier régulier**
4. Définissez le mois de début
5. Appuyez sur **Enregistrer le rôle**

## Étape 4 — Activer les rappels (optionnel)

1. Ouvrez **Paramètres → Paramètres de lecture biblique**
2. Faites défiler jusqu'à **Rappel quotidien**
3. Activez-le et choisissez une heure

## Étape 5 — Activer le suivi de série (optionnel)

1. Ouvrez **Paramètres → Paramètres de lecture biblique**
2. Activez **Série de lecture**
3. Le compteur de série apparaît dans la carte de progression de l'onglet Bible

Des célébrations de jalons apparaissent après 5, 14, 30, 90, 180 et 365 jours consécutifs.
